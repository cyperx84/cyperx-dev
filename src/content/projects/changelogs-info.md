changelogs.info aggregates and indexes changelogs from thousands of open-source projects. Subscribe to packages you depend on and get notified when breaking changes drop.

Built with a custom crawler that parses CHANGELOG.md, GitHub releases, and commit messages to extract semantic versioning data.
