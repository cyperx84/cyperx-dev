OpenClaw is an open-source robotic actuator system designed for skateboard trick execution. It uses neural networks trained on corrupted VHS skate footage to predict and execute tricks in real-time.

Features:
- Brutalist hardware feedback loop
- Claude Code integration for trick hallucination
- Zero safety checks mode (COWARDLY_MODE_OFF)
- Real-time stack trace generation on bail
