# Glitch Art Maximalist ## Product Overview **The Pitch:** A developer blog showcasing AI-generated coding projects (built with Claude Code, Codex, OpenClaw) that feels like a corrupted mainframe bleeding into a psychedelic skate video. It abandons clean, modern UX for a sensory assault of AI-generated code snippets, 3D skull and mushroom renders, and terminal-era ASCII art, making reading technical posts an act of cyberpunk survival intertwined with counter-culture board sports. **For:** Hackers, creative coders, skaters, surfers, and psychonauts who crave brutalist interfaces and despise sanitized corporate web design. **Device:** desktop **Design Direction:** Loud, chaotic glitch art style with harsh contrasts, neon red/cyan effects, overlapping elements, 3D skulls, glitchy mushroom ASCII, dense emoji clusters, and broken ASCII text featuring board-sport and AI-coding motifs. **Inspired by:** Cruelty Squad UI, vintage 90s warez terminal interfaces, 90s skate VHS tapes, cyberpunk psychonaut culture --- ## Screens - **[INDEX]:** The chaotic home screen listing the latest corrupted data blocks (AI-generated projects) - **[DATABLOCK]:** The individual article view featuring broken grids and glitching code blocks generated by LLMs - **[MANIFESTO]:** The author's about page disguised as a hacked terminal session detailing AI lore and shredding - **[FRAGMENTS]:** The archive page filtering posts by fragmented tags - **[FATAL_ERROR]:** A 404 page featuring a rotating 3D ASCII mushroom/skull and system crash logs --- ## Key Flows **Accessing a Datablock:** Reading an article from the home page 1. User is on [INDEX] -> sees overlapping list of glitching post titles over a scattered background of ASCII weed leaves and skulls 2. User clicks [Post Title] -> screen flashes with CSS inverted colors for 100ms 3. User lands on [DATABLOCK] -> content loads with typewriter effect, surrounded by ASCII noise and AI-generated terminal lore ---
Design System ## Color Palette - **Primary:** `#00FFFF` - Neon Cyan (Links, primary interactive borders) - **Background:** `#050505` - Deep Vantablack (Page background) - **Surface:** `#111111` - Charcoal (Code blocks, article containers) - **Text:** `#F8F8F8` - Blinding White (Primary body copy) - **Muted:** `#555555` - Glitch gray (ASCII art background noise, timestamps) - **Accent:** `#FF003C` - Neon Red (Hover states, glitch overlays, warnings) - **Secondary Accent:** `#FF00FF` - Magenta (Emoji cluster glows, overlapping text shadows) ## Typography - **Headings:** `Rubik Glitch`, 400, 48-72px - **Body:** `VT323`, 400, 22px (Aggressively large pixel font) - **Code/Terminal:** `Space Mono`, 400, 16px - **Buttons:** `Space Mono`, 700, 18px, uppercase **Style notes:** Hard edges only (0px border-radius). Intense use of `text-shadow: 2px 0 #FF003C, -2px 0 #00FFFF` to create constant chromatic aberration. Overlapping `div` containers with `mix-blend-mode: difference` or `exclusion`. Dense emoji clusters (💀🍄🏂🌿🛹🤖) scattered absolutely positioned across the viewport. ## Design Tokens ```css :root { --color-primary: #00FFFF; --color-background: #050505; --color-surface: #111111; --color-text: #F8F8F8; --color-muted: #555555; --color-accent: #FF003C; --color-magenta: #FF00FF; --font-heading: 'Rubik Glitch', system-ui; --font-body: 'VT323', monospace; --font-code: 'Space Mono', monospace; --radius: 0px; --spacing-sm: 8px; --spacing-md: 24px; --spacing-lg: 64px; --glitch-shadow: 3px 0px 0px #FF003C, -3px 0px 0px #00FFFF; } ```

Screen Specifications ### [INDEX] **Purpose:** Feed of the latest AI coding projects and blog posts **Layout:** Broken asymmetrical grid. Sidebar navigation locked to the left, but content blocks are intentionally offset by random 10-20px margins, overlapping the sidebar slightly. **Key Elements:** - **Navigation Console:** Left-aligned, 250px width, sticky. Links bracketed by ASCII: `[> HOME]`, `[> MANIFESTO]`. - **Post Feed:** Off-center right, stacked overlapping cards. - **Background Noise:** Fixed position `div` containing dense ASCII characters (`%#@!&`) in `#555555` at 10% opacity, covering the entire viewport. - **Floating Render:** Absolute positioned 3D skull or glitching AI-generated mushroom PNG, bottom right, rotating via CSS keyframes. **States:** - **Empty:** "ERR_GNARLY_WIPEOUT: NO_DATA_FOUND" flashing in red neon - **Loading:** Blinking cursor block `█` with "GENERATING_AI_LORE..." text - **Error:** Screen turns solid `#FF003C`, white text crash dump **Components:** - **Post Card:** `600px` width, `border: 1px solid #00FFFF`. Content offset by `-10px` upwards. Hover triggers red/cyan text shadows. - **Emoji Cluster:** Absolute positioned `span`, `z-index: 50`. E.g., `🍄🏂🤖` overlapping the corner of a post card. **Interactions:** - **Hover Post Card:** Entire card shifts `2px` left, border color switches to `#FF003C`, `text-shadow` glitch effect activates on title. - **Click Link:** CSS visual noise keyframe animation triggers on the `body` tag for 200ms. ### [DATABLOCK] **Purpose:** Reading an individual article / AI project breakdown **Layout:** Single chaotic column. 800px max-width, but images and code blocks aggressively break out of the container by 100px on either side. **Key Elements:** - **Article Header:** Massive `72px` title, uppercase, spanning multiple lines, overlapping itself with `line-height: 0.8`. - **Metadata Block:** Flashing timestamp, tag list rendered as `[FRAG: CLAUDE]`, `[FRAG: OPENCLAW]`. - **Code Snippets:** `#111111` background, `#39FF14` (Acid Green) text, `border-left: 5px solid #FF003C`. - **Inline Glitches:** Random spans of text wrapped in a `.glitch-text` class that jitter horizontally. **States:** - **Empty:** N/A (Redirect to 404) - **Loading:** Matrix-style falling green ASCII characters before content snaps into place **Components:** - **Divider:** Not a line, but a string of ASCII emojis: `🍄 🛹 🍄 AI_HALLUCINATION 🍄 🛹 🍄` in `#00FFFF`. - **Pull Quote:** Large italic `VT323` text, neon magenta drop shadow, overlapping an absolute positioned, giant transparent emoji `🌿`. **Interactions:** - **Scroll:** 3D mushroom/skull graphics in the background slowly rotate on the Y-axis based on scroll position. - **Hover Code Block:** Text color flashes from green to blinding white instantly. ### [MANIFESTO] **Purpose:** About page and author info (AI/skate/surf/psychonaut lore) **Layout:** Fake terminal window occupying 80% of the screen, centered, over a tiled background of glitching error symbols. **Key Elements:** - **Terminal Window:** `border: 2px solid #00FFFF`, inner padding `24px`, background `#050505`. - **Window Header:** "root@shredder:~", neon red background, white text. - **Content:** Author bio presented as a CLI sequence. `> whoami`, followed by bio text detailing AI experiments, snowboarding injuries, and psychedelic revelations. - **Skill List:** Rendered as loaded modules: `[OK] Claude Code...`, `[OK] Kickflips...`, `[FAIL] Sanity...`. **Interactions:** - **Load:** Text renders character by character mimicking a 56k modem connection. ### [FRAGMENTS] **Purpose:** Archive and category filtering **Layout:** A chaotic tag cloud where tags vary wildly in font-size (16px to 128px) and rotation (-15deg to 15deg). **Key Elements:** - **Tag Cloud:** Scattered links filling the entire viewport. - **Selected Tag View:** Once clicked, the cloud fades out, and a jagged, broken list of posts drops down from the top edge. **Interactions:** - **Hover Tag:** Tag scales up `1.2x`, `z-index` pops to `100`, applies magenta/cyan text shadow. ### [FATAL_ERROR] **Purpose:** 404 Page **Layout:** Full screen, completely centered, brutalist. **Key Elements:** - **Giant ASCII Art:** A massive ASCII representation of a skull morphing into a mushroom, pulsing between `#FF003C` and `#00FFFF`. - **Error Code:** `404 - BAD_TRIP / SECTOR CORRUPT` in 72px `Rubik Glitch`. - **Return Button:** Blinking `[> PADDLE BACK OUT <]` text link. **Interactions:** - **Hover Button:** ASCII art scrambles into random characters, then resolves back to the mushroom/skull when hover ends.

Build Guide **Stack:** HTML + Tailwind CSS v3 **Build Order:** 1. **[FATAL_ERROR]** - Start here. Smallest scope, but immediately establishes the aggressive glitch typography, ASCII art integration, and intense color palette. 2. **[MANIFESTO]** - Establishes the core monospace typography constraints and the terminal aesthetic for isolated content blocks. 3. **[INDEX]** - Defines the broken grid layout, overlapping elements, and the base structure for the navigation sidebar. 4. **[DATABLOCK]** - Extends index styles to long-form content, requiring custom markdown styling (prose overrides) for code blocks and pull quotes. 5. **[FRAGMENTS]** - Applies the established interactions to a chaotic, physics-like layout of tags.
